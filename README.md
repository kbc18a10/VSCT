# vsct
## 初回起動時
```
npm install
```
## それ以降
```
npm start
```
## ディレクトリ構造
```
vsct
 └ back バックエンド(Express)
 └ front フロントエンド(React)
    └ src
       └ components ヘッダーやpagesのファイルに配置するコンポーネントファイル
       └ pages ルーティングで直接参照されるファイル
       └ image 画像ファイル
       └ css cssファイル
```
## アプリケーションURL
https://vsct.herokuapp.com/
